Данная папка содержит примеры конфигурации аналитических сессий для апробации
работы программы на реальных тематиках, актуальных для российского медиапространства.

## Как использовать

1. Откройте LOM Analyzer и создайте новую сессию
2. Выберите пример из списка выше
3. Заполните поля экрана **Настройка сессии** согласно указанным значениям
4. На этапе сбора данных программа автоматически загрузит посты из VK
5. На этапе валидации темы проверьте релевантность собранных постов
6. Изучите результаты на экранах дашборда, динамики и рисков

> **Примечание:** Референсные тексты приведены как образцы реальных постов VK.
> Для наилучших результатов в режиме FULL NLP рекомендуется 5–10 текстов.
