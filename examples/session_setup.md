# Пример заполнения сессии: развитие ИИ в России

Значения ниже соответствуют набору данных [`dataset_demo.json`](dataset_demo.json)
и позволяют пройти весь пайплайн, не обращаясь к VK.

## Параметры сессии

| Поле | Значение |
|------|----------|
| **Название сессии** | Развитие ИИ в России |
| **Тематический запрос** | искусственный интеллект |
| **Регион** | *(не указывать — тема общероссийская)* |
| **Окно baseline** | 60 дней |
| **Текущее окно** | 30 дней |
| **Режим ролей** | Квадранты |

## N-граммы

Первичные — прямые обозначения темы. Совпадение по ним даёт основной вклад
в балл первого прохода.

```
ии, нейросеть, искусственный интеллект, chatgpt, gigachat, алиса, llm, модель, алгоритм, нейронка, генеративный, промпт, яндекс, сбер, копилот
```

Вторичные — сопутствующая лексика. Входят в балл с понижающим весом 0.3:
сами по себе тему не определяют, но усиливают первичное совпадение.

```
данные, обучение, технология, автоматизация, робот, код, разработчик, рынок, труд
```

Исключённые — для этой темы не задаются. Совпадение по такой n-грамме обнуляет
балл поста независимо от остальных: список нужен, когда ключевое слово многозначно
и уводит в чужую тему.

## Референсные тексты

Задают эталон темы для второго прохода фильтра: пост, попавший в пограничную зону
по ключевым словам, сравнивается с ними по смыслу (косинусная близость эмбеддингов,
порог 0.55). Вводятся построчно, по одному тексту на строку.

> Нейросети и генеративные модели меняют рынок труда в России.

> Российские компании внедряют искусственный интеллект в свои продукты.

> Эксперты предупреждают о рисках больших языковых моделей и утечке данных.

Трёх текстов достаточно для запуска; 5–10 дают более устойчивый эталон.

## Как прогнать

1. На экране **Настройка сессии** заполните поля значениями из таблицы выше.
2. Там же импортируйте [`dataset_demo.json`](dataset_demo.json) — сбор из VK
   будет пропущен, данные возьмутся из файла.
3. Запустите анализ и дождитесь завершения всех этапов.
4. Проверьте отбор постов на экране **Валидация темы**, результаты — на
   **Дашборде ЛОМ**, диагностику прогона — на экране **Качество сессии**.
