# Анкета проекта LOM Analyzer

## 1. Технологический стек проекта

### Язык и платформа
- **Kotlin 2.0** — основной язык разработки (JVM 17)
- **Compose Desktop 1.7** — UI-фреймворк (декларативный, кроссплатформенный)

### Сборка и качество
- **Gradle Kotlin DSL** — система сборки
- **Detekt** — статический анализ кода
- **JUnit 5 + Kotest + MockK** — тестирование (294 теста)

### Сеть и API
- **Ktor CIO 3.0** — HTTP-клиент для работы с VK API
- **kotlinx-serialization** — сериализация JSON

### Хранение данных
- **SQLite** (WAL-режим) — встраиваемая СУБД
- **Exposed ORM** — ORM для Kotlin
- **Flyway** — миграции схемы БД (6 миграций)

### Внедрение зависимостей
- **Koin 3.5** — DI-контейнер

### NLP и анализ текста
- **Apache Lucene (Snowball)** — стемминг/лемматизация (fallback-режим)
- **Python FastAPI sidecar** — продвинутый NLP (dostoevsky, pymorphy3, natasha, rubert-tiny2) — опциональный
- **Собственный словарный анализатор тональности** — sentilex (330+ слов), обработка отрицаний, сигнальный fallback

### Визуализация
- **Lets-Plot** — построение графиков и диаграмм

### Логирование
- **Logback** + Logstash JSON encoder — структурированное логирование

### Математический аппарат (реализован на Kotlin)
- Theil-Sen регрессия, Huber M-оценка, MAD-based R²
- Bootstrap (300×100 двухуровневый для I_base, 500 для I_event, 300 блочный для Risk)
- Robust Sigmoid нормализация, GMM-кластеризация ролей
- Z-score аномалии с сезонной декомпозицией

## 2. Оценка значимости содержимого проекта

Проект имеет **учебный характер** и разработан в рамках итогового курсового проекта.

Практическая ценность — демонстрация навыков:
- Проектирования многоэтапной аналитической системы (35-стадийный пайплайн)
- Работы с внешними API (VK API: rate limiting, backoff, пагинация)
- Построения desktop-приложения с reactive UI (Compose Desktop)
- Применения робастных статистических методов к реальным данным
- Организации кодовой базы (модульная архитектура, DI, миграции, тестирование)
- Реализации NLP-пайплайна для русского языка без внешних ML-сервисов

Проект не предназначен для коммерческого использования и не содержит конфиденциальных данных или алгоритмов, представляющих коммерческую тайну.
