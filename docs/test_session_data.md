# Тестовые данные для запуска сессии LOM Analyzer

## Название сессии

```
ИИ в России — тест
```

## Тема исследования

```
Искусственный интеллект в России
```

## Первичные n-граммы

Ядро темы — посты с любым из этих слов/фраз сразу считаются релевантными.

```
искусственный интеллект, нейросеть, нейросети, ИИ, машинное обучение, генеративный ИИ, большая языковая модель
```

## Вторичные n-граммы

Контекстные маркеры — усиливают уверенность, но сами по себе недостаточны.

```
GigaChat, YandexGPT, ChatGPT, LLM, Сбер ИИ, Яндекс нейро, цифровая экономика, импортозамещение, автоматизация, обучение модели, датасет, промпт
```

## Исключённые n-граммы

Ложные срабатывания — посты с этими фразами отсекаются даже при наличии первичных.

```
ИИ пророка, искусственное вскармливание, искусственный газон, искусственная ёлка
```

## Референсные тексты

Каждая строка — отдельный пример. Вставлять в поле целиком, без кавычек и нумерации.

```
Сбер представил обновлённую версию GigaChat с поддержкой мультимодальности и генерации изображений
Яндекс запустил YandexGPT четвёртого поколения для бизнес-клиентов и разработчиков
Минцифры предложило обязать госкомпании использовать отечественные ИИ-решения вместо зарубежных аналогов
Российские вузы массово открывают магистратуры по искусственному интеллекту и анализу данных
Эксперты обсуждают риски применения нейросетей в образовании и на рынке труда в России
```

## Сообщества

Оставить пустым — система использует newsfeed.search с автоматической сегментацией по датам.
