## 6. Структура JSON-файла

```json
{
  "communities": [
    {
      "vkId": 100001,
      "name": "Нейросети: IT, ChatGPT, ИИ",
      "screenName": "neural_networks_ai",
      "membersCount": 335000,
      "isClosed": false,
      "type": "group"
    }
  ],
  "authors": [
    {
      "vkId": 1001,
      "firstName": "Герман",
      "lastName": "Греф",
      "screenName": "gref_german",
      "followersCount": 485000,
      "isClosed": false
    }
  ],
  "posts": [
    {
      "vkId": 10001,
      "ownerId": -100001,
      "fromId": 1001,
      "date": 1779580800,
      "text": "Текст поста...",
      "likes": 3420,
      "reposts": 890,
      "comments": 567,
      "views": 185000,
      "containsMedia": false,
      "hasCopyHistory": false,
      "window": "CURRENT"
    }
  ]
}
```

### Описание полей поста:
- **vkId** — уникальный идентификатор поста
- **ownerId** — ID сообщества (отрицательный, по формату VK API)
- **fromId** — ID автора поста
- **date** — Unix timestamp (секунды)
- **text** — текст поста
- **likes/reposts/comments/views** — метрики вовлечённости
- **containsMedia** — наличие медиа-вложений
- **hasCopyHistory** — является ли пост репостом
- **window** — `BASELINE` или `CURRENT`

## 7. Цель датасета и ожидаемые результаты анализа

### 7.1. Ожидаемая квадрантная классификация

| Роль | Ожидаемые авторы | Обоснование |
|------|-----------------|-------------|
| **AUTHORITATIVE_LOM** | Греф, Ашманов, Варламов, Касперская | Высокий I_base (большая аудитория + engagement) + высокий I_event (активность в текущем окне) |
| **SLEEPING_GIANT** | Воронцов, Крайнов, Инвестиров | Высокий I_base, но низкий I_event (публикуют редко в текущем окне) |
| **TOPIC_DRIVER** | Врачёва, Фрилансова, Геймеров | Низкий I_base (маленькая аудитория), но высокий I_event (активно пишут по теме) |
| **BACKGROUND** | Боты, репостеры, пассивные | Низкий I_base + низкий I_event |

### 7.2. Ожидаемые аномалии

- **Volume Spikes** — дни с концентрацией постов от экспертов (несколько авторов публикуют в один день)
- **Giant Activation** — активизация «спящих гигантов» в текущем окне

### 7.3. Тестируемые возможности системы

| Компонент | Что тестируется |
|-----------|----------------|
| Topic Filtering | Отсечение off-topic постов (дача, игры) |
| Deduplication | Обнаружение повторяющихся ботов-текстов |
| Role Classification | Разделение на 4 квадранта |
| Anomaly Detection | Всплески активности |
| Risk Scoring | Оценка координированности |
| Sentiment | Разброс тональности (позитив, негатив, нейтрал) |
| JSON Import | Работа пайплайна без VK API |
